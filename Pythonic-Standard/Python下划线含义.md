通常会见到5种下划线的情况



|        类型        |   表示    |                        含义                        |
| :----------------: | :-------: | :------------------------------------------------: |
|    单前导下划线    |  `_var`   | 一种约定俗成的，解释器不会有额外操作，仅供内部使用 |
|    单末尾下划线    |  `var_`   |              为避免和python关键字冲突              |
|    双前导下划线    |  `__var`  |                     名称修饰器                     |
| 双前导和末尾下划线 | `__var__` |        python一些自定义的方法，自己不要使用        |
|      单下划线      |    `_`    |                 临时无意义的空变量                 |
